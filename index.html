🎖️ Outil de renommage OTAN (détection de tous les tableaux)

Options de base

Format: TRIGRAMME=noms_séparés_par_des_virgules

Format: BRANCHE=mot1,mot2,mot3 (séparés par des virgules)

Variables disponibles: - {COUNTRY} : Trigramme du pays (ex: ALB) - {BRANCH} : Branche militaire (AIR, LAND, NAVY) - {RANK} : Grade OTAN (ex: OF8) - {LATIN_NAME} : Nom du grade avec des tirets (ex: Gjeneral-lejtant) - {ORIGINAL} : Nom original du fichier (sans extension) - {EXT} : Extension du fichier (ex: .svg, .png)

Intégration Wikipedia (API MediaWiki)

Exemple: https://en.wikipedia.org/wiki/Comparative_military_ranks_of_NATO

Exemple: table (pour cibler tous les tableaux) ou .wikitable (pour les tableaux Wikipedia)
